# Pika Storm Run

Open [index.html](/Users/anaafonso/Desktop/pika-storm-run/index.html) in a browser to play.

If your browser blocks local file access, run a simple local server from this directory:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

Notes:

- The game is self-contained and uses code-drawn art only.
- It includes 25 stages, three playable Pokemon, storm collectibles, skill motes, and shadow enemies.
- It uses simple code-drawn visuals instead of external assets.
