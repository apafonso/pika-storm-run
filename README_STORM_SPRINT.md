# Pokemon Comet Quest

Open [index.html](/Users/anaafonso/Desktop/pika-storm-run/index.html) in a browser to play.

If your browser blocks local file access, run a simple local server from this directory:

```bash
cd /Users/anaafonso/Desktop/pika-storm-run
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

Highlights:

- 100 stages across 10 worlds.
- 4 playable Pokemon: Pikachu, Bulbasaur, Charmander, and Squirtle.
- Character choice locks for the full campaign until you start a new run.
- Different powers, pickups, and movement styles for each character.
- Commander Keen style progression with comet parts, boss lairs, and campaign upgrades.
- Bosses include Meowth, Lapras, Charizard, and a final Mewtwo fight.
